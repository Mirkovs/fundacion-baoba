# Imágenes necesarias para la Fundación Baobab

Este documento lista las imágenes que deben agregarse al proyecto. Todas las imágenes deben colocarse en la carpeta `public/images/`.

## Imágenes requeridas

### Hero Section
- **baobab-tree.jpg** - Imagen del árbol Baobab (visible en la captura original)
  - Dimensiones recomendadas: 800x600px mínimo
  - Ubicación en captura: Parte superior izquierda de la página original

### Programas
- **students.jpg** - Estudiantes becados en clase
  - Dimensiones recomendadas: 600x400px
  - Ubicación en captura: Sección "La fundación" - imagen de estudiantes

- **baobab-junior.jpg** - Grupo de miembros de Baobab Junior
  - Dimensiones recomendadas: 600x400px
  - Ubicación en captura: Sección "Asociación Baobab Junior"

### Proyectos
- **establo.jpg** - Construcción del establo
  - Dimensiones recomendadas: 400x300px
  - Ubicación en captura: Sección "Proyectos desarrollados"

- **caci.jpg** - Centro educativo CACI / Maison Baobab
  - Dimensiones recomendadas: 400x300px
  - Ubicación en captura: Sección "La fundación" - placa CACI

- **community.jpg** - Proyectos comunitarios
  - Dimensiones recomendadas: 400x300px
  - Ubicación en captura: Imagen de actividades comunitarias

### Otras secciones
- **foundation.jpg** - Imagen general de la fundación
  - Dimensiones recomendadas: 600x400px
  - Puede ser cualquier imagen representativa de las actividades

- **news-1.jpg** - Imagen para noticias
  - Dimensiones recomendadas: 600x400px
  - Ubicación en captura: Sección "Noticias"

## Instrucciones

1. Extrae las imágenes de la captura original proporcionada
2. Guárdalas con los nombres exactos listados arriba
3. Colócalas en `public/images/`
4. Optimiza las imágenes para web (formato WebP recomendado para mejor rendimiento)

## Alternativa temporal

Si no tienes acceso inmediato a las imágenes, puedes:
1. Usar placeholders temporales con las dimensiones recomendadas
2. Reemplazarlas más tarde con las imágenes reales

## Formato recomendado

- Formato: JPG o WebP
- Calidad: 80-85% para balance entre calidad y tamaño
- Optimización: Usar herramientas como TinyPNG o Squoosh
